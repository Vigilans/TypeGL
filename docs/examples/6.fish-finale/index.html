Keyboard Control Guidance:

1: Control Fish_kun
2: Control Fish_aqua
3: Control Light Source

Oriented Objects (e.g. Fish):
Q/E: Scale Up/Scale Down
W/S: Move fowared/Move backward
A/D: Rotate along normal
←/→: Rotate along orientation
↑/↓: Rotate along normal × orient

Simple Objects (e.g. Light Source):
Q/E: Scale Up/Scale Down
←/→: Move by X axis
↑/↓: Move by Y axis
W/S: Move by Z axis
A/D: Rotate along Y axis

Camera：
Left Click: Rotate along focus ball
Middle Click: Move along diameter
Right Click: Move along tanget plane

(Focus ball refers to the sphere whose heart set on focus point, with a radius of the distance between the foucs point and the camera)
